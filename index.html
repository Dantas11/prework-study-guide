HTML

  • The head element contains informations about the webpage.
  • The body element represents the visible content shown to the user.
  • The < ul > element represent unordered lists. The < ol > element represent ordered lists. The < il > element represent list items.
  • The `src` attribute defines the location, or source, of the image file. The `alt` attribute contains a text string that describes the appearance and functionality of the image.

Git

  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switchs to it.
  • git add -A adds modifications in the current working branch. -A flag indicates that we want to add all changes.
  • git commit -m commit all changes to our repository.
  • Feature branch is created as which a copy of the "main" branch so we can make changes. Then we push up the changes to our feature brench in our GitHub repository. PR "Pull Request" is used to merge changes into our the base branch. Once the PR is opened reviwed, checked by others and approved, you can merde it into the main branch.
