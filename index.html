HTML

  • The head element contains informations about the webpage.
  • The body element represents the visible content shown to the user.
  • The < ul > element represent unordered lists. The < ol > element represent ordered lists. The < il > element represent list items.
  • The `src` attribute defines the location, or source, of the image file. The `alt` attribute contains a text string that describes the appearance and functionality of the image.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A pading indicates how much space we want around the content inside an element.
  • The period preceding < card > designates this as a class selector. Any HTML element assigned to the class,< card >, will gain these property assigments from the CSS.

Git

  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switchs to it.
  • git add -A adds modifications in the current working branch. -A flag indicates that we want to add all changes.
  • git commit -m commit all changes to our repository.
  • Feature branch is created as which a copy of the "main" branch so we can make changes. Then we push up the changes to our feature brench in our GitHub repository. PR "Pull Request" is used to merge changes into our the base branch. Once the PR is opened reviwed, checked by others and approved, you can merde it into the main branch.
  • Git workflow steps to add, commit, and push your work to your feature branch. Then open a PR.

JavaScript

  • Control flow is the order in wich a computer executes code in a file. JavaScript is executed from top to bottom.
  • Using conditional staments such as < if >, allows to interrupt control flow.
  • console.log method to output a message to the web console by adding an argument.
  • The console.log method was created to make it easier for developers to test their code. Developers use it daily to debug and test the code that they write.
  • Array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
